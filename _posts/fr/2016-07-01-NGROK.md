ngrok expose au public, via des tunnels sécurisés, des serveurs locaux situés derrière des NAT et des pare-feux.

<!--more-->

Commencez par télécharger le fichier exe à cet emplacement : https://ngrok.com/download

Extrayez l'application, ouvrez une console et allez dans le répertoire où ngrok a été extrait.

Lancez l'application ngrok.exe et ajoutez le port à exposer :

{% highlight c %}

ngrok http 8081

{% endhighlight %}

De plus, si vous utilisez IIS Express avec Visual Studio, vous devez effectuer une configuration supplémentaire pour atteindre votre application.

* Modifiez le fichier applicationhost.config (dans le dossier : [VSPROJECTFOLDER]/.vs/config/ ),
* Trouvez la ligne contenant "*:8081:localhost" et supprimez ":localhost" pour ne conserver que "*:8081".
* Redémarrez votre Visual Studio.

* Appelez ngrok avec :
{% highlight c %}

ngrok http 8081 -host-header="localhost:8081"

{% endhighlight %}
 Une fois ngrok lancé, vous accédez à votre tunnel via une URL, c'est-à-dire : http://1cac6d41.ngrok.io
